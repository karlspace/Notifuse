## How to Translate a File

1. Add `useLingui` import:
   ```tsx
   import { useLingui } from '@lingui/react/macro'
   ```

2. Get the `t` function in component:
   ```tsx
   const { t } = useLingui()
   ```

3. Wrap strings with template literals:
   ```tsx
   // Before
   <Button>Save</Button>

   // After
   <Button>{t`Save`}</Button>
   ```

4. Run extract to update PO files:
   ```bash
   npm run lingui:extract
   ```

5. Translate strings in `src/i18n/locales/{locale}.po`

6. Compile for production:
   ```bash
   npm run lingui:compile
   ```

---

## Notes

- Focus on user-facing strings only (labels, buttons, messages, errors)
- Skip internal/debug strings
- Skip strings that are the same in all languages (brand names, technical terms)
- Consider context when translating - same English word may need different translations
